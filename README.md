## 功能介绍
- 前端应用开发完后打包后自助上传部署
- 配合服务端脚本实现服务端渲染SSR功能
- 可以快速回滚到上一个版本
- 可以设置环境变量供SSR功能使用
- 服务端脚本提供执行日志查看功能
